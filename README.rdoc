= Freifunk KBU Info Page

This is the source code for the {Freifunk Köln, Bonn und Umgebung}[http://kbu.freifunk.net] information {page}[http://info.kbu.freifunk.net]. The page containts basic inforamtion to our Freifunk project and is stored on access points. It based on the static web site generator {nanoc}[http://nanoc.stoneship.org/].


== Build website

0. You need to have Ruby and the Bundler gem installed.

1. Clone repository

     clone git@github.com:meise/freifunk-kbu-info-page.git

2. Bundle inside directory to install all necessary gems.

     cd freifunk-kbu-info-page
     bundle

3. Build static page. Static files are located in output directory after compilation.

     nanoc compile
